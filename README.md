| Fugue | [赋格](https://en.wikipedia.org/wiki/Fugue) | 复调音乐的重要曲式，同一旋律反复出现，互相重叠，或互相“逃难”对方 |
| Gavotte | [加沃特](https://en.wikipedia.org/wiki/Gavotte) |  起源于法国民间的中速、四拍子舞曲 |

| Humoresque | [幽默曲](https://en.wikipedia.org/wiki/Humoresque) | 段落的表情极富变化 |

| Invention| [创意曲](https://en.wikipedia.org/wiki/Invention_(musical_composition))| 复调结构的钢琴小曲。类似前奏曲或小赋格曲。巴赫始用此名称 |

| Lullaby | [摇篮曲](https://en.wikipedia.org/wiki/Lullaby)  | 抚人心灵的乐曲，一般是希望以熟悉的声音和甜美的旋律用作哄小孩入睡或停止哭闹 |
| Magrigal | [牧歌](https://en.wikipedia.org/wiki/Madrigal) | 没有乐器伴奏的声乐作品 |
| March | [进行曲](https://en.wikipedia.org/wiki/March_(music)) | 为进行或游行而作的具有强烈的音乐节奏的短曲 |
| Mass | [弥撒曲](https://en.wikipedia.org/wiki/Mass_(music)) | 天主教、东正教和圣公会的弥撒仪式所用声乐套曲 |
| Mazurka | [玛祖卡](https://en.wikipedia.org/wiki/Mazurka) | 起源于波兰民间的情绪活泼热烈的三拍子双人舞曲 |
| Minuet | [小步舞曲](https://en.wikipedia.org/wiki/Minuet) | 源于法国的三拍子舞曲，男女对舞 |
| Monophony | [单音](https://en.wikipedia.org/wiki/Monophony) | 只有一条旋律线的音乐 |
| Motet | [经文歌](https://en.wikipedia.org/wiki/Motet) | 中世纪到文艺复兴时期的一种宗教音乐体裁 |
| Nocturne | [夜曲](https://en.wikipedia.org/wiki/Nocturne) | 形式自由的三段体器乐短曲 |
| Novelette | [新事曲](https://en.wikipedia.org/wiki/Novelette_(music)) | 始创于R.舒曼的一种钢琴特性曲 |
| Opera | [歌剧](https://en.wikipedia.org/wiki/Opera) | 主要或完全以歌唱和音乐来交代和表达剧情的戏剧 |

| Overture | [序曲](https://en.wikipedia.org/wiki/Overture) | 歌剧、舞剧等开幕前演奏的短曲，亦称"开场音乐"  |
| Partita | [帕蒂塔](https://en.wikipedia.org/wiki/Partita) | 原为单乐器片段，巴洛克时期被用作舞蹈套曲 |
| Polka | [波尔卡](https://en.wikipedia.org/wiki/Polka) | 起源于捷克的速度较快的二拍子舞曲 |
| Polonaise | [波罗乃兹](https://en.wikipedia.org/wiki/Polonaise) | 波兰舞曲,起源于波兰民间的庄重，缓慢的三拍子舞曲 |
| Polyphony| [复调音乐](https://en.wikipedia.org/wiki/Polyphony) | 两条以上（含）独立旋律，通过技术性处理，和谐地结合在一起 |
| Prelude | [前奏曲](https://en.wikipedia.org/wiki/Prelude_(music)) | 放在各幕之前的短乐曲 |
| Perpetuum mobile | [无穷动](https://en.wikipedia.org/wiki/Perpetuum_mobile) | 从头至尾贯穿着急速节奏的乐曲 |
| Recitative | [宣叙调](https://en.wikipedia.org/wiki/Recitative) | 歌剧、清唱剧、康塔塔等大型声乐中类似朗诵的曲调 |
| Requiem | [安魂弥撒](https://en.wikipedia.org/wiki/Requiem) | 天主教礼仪为死者送葬的弥撒曲 |
| Rhapsody | [狂想曲](https://en.wikipedia.org/wiki/Rhapsody_(music)) | 出现于19世纪初的一种富有民族特色的史诗性器乐曲，常直接采用民间曲调 |
| Romance | [浪漫曲](https://en.wikipedia.org/wiki/Romance_(music)) | 无固定形式的抒情短歌或短曲 |
| Rondo | [回旋曲](https://en.wikipedia.org/wiki/Rondo) | 起源于欧洲民间的轮舞歌曲，每唱一段后就重复副歌 |
| Round| [轮唱曲](https://en.wikipedia.org/wiki/Round_(music)o) | 用同度或八度卡农手法写成的声乐曲 |
| Scherzo | [诙谐曲](https://en.wikipedia.org/wiki/Scherzo) | 快速，节奏强烈的器乐曲，一般为三拍子，有令人惊奇、幽默的特点 |
| Serenade | [小夜曲](https://en.wikipedia.org/wiki/Serenade) | 一种源自表达爱情的欢快明朗典雅质朴的如歌乐曲 |
| Sonata | [奏鸣曲](https://en.wikipedia.org/wiki/Sonata) | 汇源自拉丁文的sonare，即发出声响，有别于歌唱的声音 |
| Songs Without Words| [无言歌](https://en.wikipedia.org/wiki/Songs_Without_Words) | 门德尔松创作的一系列短篇抒情钢琴作品 |
| Suite | [组曲](https://en.wikipedia.org/wiki/Suite_(music)) | 套曲形式的器乐曲或交响曲，即各自独立的不同乐曲的组合 |
| Symphonic poem | [交响诗](https://en.wikipedia.org/wiki/Symphonic_poem) | 与Tone Poem（音诗）同义。通常为单乐章，具有叙事性，或者呈现出故事性 |
| Symphony | [交响曲](https://en.wikipedia.org/wiki/Symphony) | 包含多个乐章的大型管弦乐曲，一般是为管弦乐团创作 |
| Tango | [探戈](https://en.wikipedia.org/wiki/Tango) | 由非洲牧童所发明的阿根廷双人舞蹈 |
| Tarantella | [塔朗泰拉](https://en.wikipedia.org/wiki/Tarantella) | 起源于意大利的速度迅急、情绪热烈的舞曲 |
| Toccata | [触技曲](https://en.wikipedia.org/wiki/Toccata) | 通常由键盘乐器或拨弦乐器独奏的乐曲，具有耀技性 |
| Variation | [变奏曲](https://en.wikipedia.org/wiki/Variation_(music)) | 将乐曲最初呈示的主题不断地反复，在反复的过程中有变化 |
| Waltz | [华尔兹](https://en.wikipedia.org/wiki/Waltz) | 圆舞曲,一种起源于奥地利民间的三拍子舞曲 |
